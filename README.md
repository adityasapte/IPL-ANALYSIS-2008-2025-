## 🏏 IPL Analysis Dashboard (2008–2025)

### 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of IPL seasons from 2008 to 2025. It highlights team performance, player achievements, match statistics, and seasonal insights.

---

### 📊 Key Features

* Season Winner & Runner-Up
* Orange Cap Holder (Top Run Scorer)
* Purple Cap Holder (Top Wicket Taker)
* Most Fours & Most Sixes
* Total Matches, Venues & Teams
* Points Table Analysis
* Season-wise filtering using slicer

---

### 🛠 Tools & Technologies Used

* Power BI
* DAX
* Data Modeling
* Excel / CSV Dataset
* Data Cleaning & Transformation

---

### 📈 KPIs Included

* Total 6s
* Total 4s
* Total Matches
* Centuries
* Half Centuries
* Total Teams
* Total Venues

---

### 🧠 DAX Concepts Used

* CALCULATE
* FILTER
* SUMMARIZE
* USERELATIONSHIP
* SELECTEDVALUE
* LOOKUPVALUE

---


### 🎯 Business Insights

* Identifies season winners and runner-ups.
* Highlights top-performing batsmen and bowlers.
* Compares team performance using points table.
* Enables season-wise trend analysis.
